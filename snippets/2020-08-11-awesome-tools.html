# 笔记

  • Quiver #
    • 除了只有一级目录以外没啥大毛病
  • Notion
  • Trilium
    • 开源（基于 electron）
    • 能自行部署同步服务器，还提供了 web 客户端
    • 无限目录
    • 似乎主打个人知识库，笔记之间的关系管理做得很好
    • 不支持 Latex 公式（所以没用了，好肤浅的理由 2333）
  • VNote
    • 开源（基于 Qt）
    • 无限目录
    • 支持公式，不过渲染似乎有点慢...
  • OneNote
    • 巨硬系产品对公式的支持都挺令人头大...
  • 有道云笔记
  • EverNote
