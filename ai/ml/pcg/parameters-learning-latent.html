# 含隐变量的图模型的参数学习（EM 算法）

有些时候 XXX 中的变量有很复杂的依赖关系，直接建模 p(x)p(x)p(x) 会很困难，这时通常会引入隐变量 zzz 来简化模型。如果图模型中包含隐变量，即有部分变量是不可观测的，这时就需要用 EM 算法（Expectation Maximum，期望最大化算法）来进行参数估计。

# 边缘似然

令 XXX 为可观测变量集合，ZZZ 为隐变量集合。由于隐变量不可观测，因此一般改用边缘分布（也就是显变量的分布）的最大似然为目标函数。

样本 xxx 的边缘似然函数（Marginal Likelihood）为：

pθ(x)=∑zpθ(x,z)=∑zpθ(x∣z)pθ(z)p_{\theta}(x) = \sum_z p_{\theta}(x,z) = \sum_z p_\theta(x \mid z) p_\theta(z) pθ​(x)=z∑​pθ​(x,z)=z∑​pθ​(x∣z)pθ​(z)

边缘似然也称为证据（evidence）。

给定 NNN 个训练样本 D={x(n)}n=1ND=\{x^{(n)}\}^N_{n=1}D={x(n)}n=1N​，其对数边缘似然函数为：

L(D;θ)=1N∑n=1Nlog⁡pθ(x(n))=1N∑n=1Nlog⁡∑zpθ(x(n),z)=1N∑n=1Nlog⁡∑zpθ(x(n)∣z)pθ(z)\begin{aligned} L(D; \theta) &= \frac{1}{N} \sum_{n=1}^N \log p_{\theta}(x^{(n)}) \\ &= \frac{1}{N} \sum_{n=1}^N \log \sum_z p_{\theta}(x^{(n)}, z) \\ &= \frac{1}{N} \sum_{n=1}^N \log \sum_z p_\theta(x^{(n)} \mid z) p_\theta(z) \end{aligned} L(D;θ)​=N1​n=1∑N​logpθ​(x(n))=N1​n=1∑N​logz∑​pθ​(x(n),z)=N1​n=1∑N​logz∑​pθ​(x(n)∣z)pθ​(z)​

# ELBO

上式第三步意味着我们要对所有可能的 zzz 求和（或积分），除非 pθ(x,z)p_{\theta}(x, z)pθ​(x,z) 的形式非常简单，否则这在很多情况下是 intractable 的。因此，为了计算 log⁡pθ(x)\log p_{\theta}(x)logpθ​(x)，我们引入一个额外的变分函数（variational function）q(z)q(z)q(z)，q(z)q(z)q(z) 是一个定义在隐变量 ZZZ 上的分布：

log⁡pθ(x)=log⁡∑zq(z)pθ(x,z)q(z)=log⁡Eq(z)[pθ(x,z)q(z)]≥Eq(z)[log⁡pθ(x,z)q(z)]（詹森不等式）=∑zq(z)log⁡pθ(x,z)q(z)≜ELBOθ(q,x)\begin{aligned} \log p_\theta(x) &= \log \sum_z q(z) \frac{p_\theta(x,z)}{q(z)} \\ &= \log \mathbb{E}_{q(z)} [ \frac{p_\theta(x,z)}{q(z)}] \geq \mathbb{E}_{q(z)} [ \log \frac{p_\theta(x,z)}{q(z)}] \qquad \text{（詹森不等式）}\\ &= \sum_z q(z) \log \frac{p_\theta(x,z)}{q(z)} \\ &\triangleq \text{ELBO}_\theta(q,x) \end{aligned} logpθ​(x)​=logz∑​q(z)q(z)pθ​(x,z)​=logEq(z)​[q(z)pθ​(x,z)​]≥Eq(z)​[logq(z)pθ​(x,z)​]（詹森不等式）=z∑​q(z)logq(z)pθ​(x,z)​≜ELBOθ​(q,x)​

ELBOθ(q,x)\text{ELBO}_\theta(q,x)ELBOθ​(q,x) 为样本 xxx 的对数边缘似然函数 log⁡pθ(x)\log p_\theta(x)logpθ​(x) 的下界，称为证据下界（Evidence Lower Bound，ELBO）。

其中，詹森不等式（Jensen Inequlity）指，对于下凸函数 ggg，『期望的函数大于等于函数的期望』一定成立，即：

g(E[X])≥E[g(X)]g(\mathbb{E}[X]) \geq \mathbb{E}[g(X)] g(E[X])≥E[g(X)]

当且仅当 q(z)=pθ(z∣x)q(z) = p_\theta(z \mid x)q(z)=pθ​(z∣x) 时，等号成立，即 log⁡pθ(x)=ELBOθ(q,x)\log p_\theta(x) = \text{ELBO}_\theta(q,x)logpθ​(x)=ELBOθ​(q,x)。

证明过程如下（《神经网络与深度学习》习题 11-4，不想看证明的话跳过就好）：

显然，当且仅当 pθ(x,z)q(z)\frac{p_\theta(x,z)}{q(z)}q(z)pθ​(x,z)​ 的比值为一个常数时，等号成立：

pθ(x,z)q(z)=c(1)\frac{p_\theta(x,z)}{q(z)} = c \tag{1} q(z)pθ​(x,z)​=c(1)

⇒pθ(x,z)=c⋅q(z)\rArr p_\theta(x,z) = c \cdot q(z) ⇒pθ​(x,z)=c⋅q(z)

⇒∫zpθ(x,z)dz=∫zc⋅q(z)dz\rArr \int_z p_\theta(x,z) dz = \int_z c \cdot q(z) dz ⇒∫z​pθ​(x,z)dz=∫z​c⋅q(z)dz

其中 ∫zpθ(x,z)dz\int_z p_\theta(x,z) dz∫z​pθ​(x,z)dz 可以看作是 pθ(x)p_\theta(x)pθ​(x) 的边缘概率；∫zc⋅q(z)dz\int_z c \cdot q(z) dz∫z​c⋅q(z)dz 可以看作是 ccc 的边缘概率，从而：

pθ(x)=c(2)p_\theta(x) = c \tag{2} pθ​(x)=c(2)

将式 (2) 带入式 (1) 得：

q(z)=pθ(x,z)pθ(x)=pθ(z∣x)(3)q(z) = \frac{p_\theta(x,z)}{p_\theta(x)} = p_\theta(z \mid x) \tag{3} q(z)=pθ​(x)pθ​(x,z)​=pθ​(z∣x)(3)

注意，式 (3) 中的 θ\thetaθ 是一个常数值。比如当 EM 算法的第 ttt 步 arg⁡max⁡θtpθt+1(x)\arg \max_{\theta_t} p_{\theta_{t+1}}(x)argmaxθt​​pθt+1​​(x) 时，式 (3) 中的 θ\thetaθ 就是 t−1t-1t−1 步时的参数 θt\theta_tθt​。

这样，最大化对数边缘似然函数 log⁡pθ(x)\log p_\theta(x)logpθ​(x) 的过程可以分解为两个步骤：

  1. 找到近似分布 q(z)q(z)q(z) 使得 log⁡pθ(x)=ELBOθ(q,x)\log p_\theta(x) = \text{ELBO}_\theta(q,x)logpθ​(x)=ELBOθ​(q,x)

  2. 寻找能最大化 ELBOθ(q,x)\text{ELBO}_\theta(q,x)ELBOθ​(q,x) 的参数 θ\thetaθ

这就是 EM 算法。

# EM 算法

EM 算法具体分为 E 步（expectation step）和 M 步（maximization step），这两步不断重复，通过迭代的方法来最大化边缘似然。在第 ttt 步更新时，E 步和 M 步分别为：

  1. E 步：固定参数 θt\theta_tθt​ ，找到一个分布 qt+1(z)q_{t+1} (z)qt+1​(z) 使得证据下界 ELBOθt(q,x)\text{ELBO}_{\theta_t}(q,x)ELBOθt​​(q,x) 等于 log⁡pθt(x)\log p_{\theta_t}(x)logpθt​​(x)

  2. M 步：固定 qt+1(z)q_{t+1} (z)qt+1​(z)，找到一组参数使得证据下界 ELBOθt(qt+1,x)\text{ELBO}_{\theta_t}(q_{t+1},x)ELBOθt​​(qt+1​,x) 最大，即：

θt+1=arg⁡max⁡θELBOθ(qt+1,x)=arg⁡max⁡θ∑zqt+1(z)log⁡pθ(x,z)qt+1(z)=arg⁡max⁡θ∑zpθt(z∣x)log⁡pθ(x,z)pθt(z∣x)=arg⁡max⁡θ∑zpθt(z∣x)log⁡pθ(x,z)\begin{aligned} \theta_{t+1} &= \arg \max_\theta \text{ELBO}_\theta(q_{t+1},x) \\ &= \arg \max_\theta \sum_z q_{t+1}(z) \log \frac{p_\theta(x,z)}{q_{t+1}(z)} \\ &= \arg \max_\theta \sum_z p_{\theta_t}(z \mid x) \log \frac{p_\theta(x,z)}{p_{\theta_t}(z \mid x)} \\ &= \arg \max_\theta \sum_z p_{\theta_t}(z \mid x) \log p_\theta(x,z) \end{aligned} θt+1​​=argθmax​ELBOθ​(qt+1​,x)=argθmax​z∑​qt+1​(z)logqt+1​(z)pθ​(x,z)​=argθmax​z∑​pθt​​(z∣x)logpθt​​(z∣x)pθ​(x,z)​=argθmax​z∑​pθt​​(z∣x)logpθ​(x,z)​

θt\theta_tθt​ 为上一时刻的参数，pθt(z∣x)p_{\theta_t}(z \mid x)pθt​​(z∣x) 是 zzz 的后验分布。

# 从 KL 散度来理解

对数边缘似然 log⁡pθ(x)\log p_\theta(x)logpθ​(x) 可以分解为：

pθ(x,z)=pθ(z∣x)pθ(x)p_\theta(x, z) = p_\theta(z \mid x) p_\theta(x) pθ​(x,z)=pθ​(z∣x)pθ​(x)

⇒log⁡pθ(x,z)=log⁡pθ(z∣x)+log⁡pθ(x)\rArr \log p_\theta(x, z) = \log p_\theta(z \mid x) + \log p_\theta(x) ⇒logpθ​(x,z)=logpθ​(z∣x)+logpθ​(x)

⇒log⁡pθ(x)=log⁡pθ(x,z)−log⁡pθ(z∣x)\rArr \log p_\theta(x) = \log p_\theta(x, z) - \log p_\theta(z \mid x) ⇒logpθ​(x)=logpθ​(x,z)−logpθ​(z∣x)

两边同时对隐变量分布 q(z)q(z)q(z) 求期望，左边：

∑zq(z)log⁡pθ(x)=log⁡pθ(x)∑zq(z)=log⁡pθ(x)\sum_z q(z) \log p_\theta(x) = \log p_\theta(x) \sum_z q(z) = \log p_\theta(x) z∑​q(z)logpθ​(x)=logpθ​(x)z∑​q(z)=logpθ​(x)

右边可以先写成：

log⁡pθ(x,z)−log⁡pθ(z∣x)\log p_\theta(x, z) - \log p_\theta(z \mid x) logpθ​(x,z)−logpθ​(z∣x)

=(log⁡pθ(x,z)−log⁡q(z))−(log⁡pθ(z∣x)−log⁡q(z))= (\log p_\theta(x, z) - \log q(z)) - (\log p_\theta(z \mid x) - \log q(z)) =(logpθ​(x,z)−logq(z))−(logpθ​(z∣x)−logq(z))

=log⁡pθ(x,z)q(z)−log⁡pθ(z∣x)q(z)= \log \frac{p_\theta(x, z)}{q(z)} - \log \frac{p_\theta(z \mid x)}{q(z)} =logq(z)pθ​(x,z)​−logq(z)pθ​(z∣x)​

则右边对隐变量分布 q(z)q(z)q(z) 求期望：

∑zq(z)(log⁡pθ(x,z)−log⁡pθ(z∣x))\sum_z q(z) (\log p_\theta(x, z) - \log p_\theta(z \mid x)) z∑​q(z)(logpθ​(x,z)−logpθ​(z∣x))

=∑zq(z)log⁡pθ(x,z)q(z)−∑zq(z)log⁡pθ(z∣x)q(z)= \sum_z q(z) \log \frac{p_\theta(x, z)}{q(z)} - \sum_z q(z) \log \frac{p_\theta(z \mid x)}{q(z)} =z∑​q(z)logq(z)pθ​(x,z)​−z∑​q(z)logq(z)pθ​(z∣x)​

=ELBOθ(q,x)+KL(q(z)∥pθ(z∣x))= \text{ELBO}_\theta(q,x) + \text{KL}(q(z) \| p_\theta(z \mid x)) =ELBOθ​(q,x)+KL(q(z)∥pθ​(z∣x))

合起来：

log⁡pθ(x)=ELBOθ(q,x)+KL(q(z)∥pθ(z∣x))\log p_\theta(x) = \text{ELBO}_\theta(q,x) + \text{KL}(q(z) \| p_\theta(z \mid x)) logpθ​(x)=ELBOθ​(q,x)+KL(q(z)∥pθ​(z∣x))

其中，KL(q(z)∥pθ(z∣x))\text{KL}(q(z) \| p_\theta(z \mid x))KL(q(z)∥pθ​(z∣x)) 为隐变量分布 q(z)q(z)q(z) 和后验分布 pθ(z∣x)p_\theta(z \mid x)pθ​(z∣x) 的 KL 散度。

KL 散度一定 ≥0\geq 0≥0，且当且仅当 q(z)=pθ(z∣x)q(z) = p_\theta(z \mid x)q(z)=pθ​(z∣x) 时，KL(q(z)∥pθ(z∣x))=0\text{KL}(q(z) \| p_\theta(z \mid x)) = 0KL(q(z)∥pθ​(z∣x))=0，从而使得 ELBOθ(q,x)=log⁡pθ(x)\text{ELBO}_\theta(q,x) = \log p_\theta(x)ELBOθ​(q,x)=logpθ​(x)。

所以 ELBOθ(q,x)\text{ELBO}_\theta(q,x)ELBOθ​(q,x) 为 log⁡pθ(x)\log p_\theta(x)logpθ​(x) 的一个下界。因此当逐步提高这个下界时，相当于增大了 log⁡pθ(x)\log p_\theta(x)logpθ​(x)，所以要对 ELBO 求期望最大化：

θ^=arg⁡max⁡θELBOθ(q,x)\hat{\theta} = \arg \max_\theta \text{ELBO}_\theta(q, x) θ^=argθmax​ELBOθ​(q,x)

# 直觉上的证明

假设在第 ttt 步时的模型参数为 θt\theta_tθt​。

  • E 步：找到一个分布 qt+1(z)q_{t+1}(z)qt+1​(z) 使得 log⁡pθt(x)=ELBOθt(qt+1,x)\log p_{\theta_t}(x) = \text{ELBO}_{\theta_t}(q_{t+1},x)logpθt​​(x)=ELBOθt​​(qt+1​,x)

  • M 步：固定 qt+1(z)q_{t+1} (z)qt+1​(z)，找到一组参数 θt+1\theta_{t+1}θt+1​ 使得 ELBOθt+1(qt+1,x)\text{ELBO}_{\theta_{t+1}}(q_{t+1}, x)ELBOθt+1​​(qt+1​,x) 最大，则有 ELBOθt+1(qt+1,x)≥ELBOθt(qt+1,x)\text{ELBO}_{\theta_{t+1}}(q_{t+1}, x) ≥ \text{ELBO}_{\theta_t}(q_{t+1}, x)ELBOθt+1​​(qt+1​,x)≥ELBOθt​​(qt+1​,x)

因此有：

log⁡pθt+1(x)≥ELBOθt+1(qt+1,x)≥ELBOθt(qt+1,x)=log⁡pθt(x)\log p_{\theta_{t+1}}(x) \geq \text{ELBO}_{\theta_{t+1}}(q_{t+1}, x) \geq \text{ELBO}_{\theta_t}(q_{t+1}, x) = \log p_{\theta_t}(x) logpθt+1​​(x)≥ELBOθt+1​​(qt+1​,x)≥ELBOθt​​(qt+1​,x)=logpθt​​(x)
