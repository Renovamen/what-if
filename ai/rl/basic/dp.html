# 基于模型的动态规划

对于 model-based 方法而言，环境 R(s,a)R(s, a)R(s,a) 和 Pss′aP_{ss'}^aPss′a​ 是已知的，那么可以直接根据贝尔曼方程用动态规划来计算更新策略。整个过程分为策略评估（policy evaluation）和策略改进（policy improvement）两部分：

  1. 策略评估：评估每个状态的 VVV 值函数

Vπ(s)=∑a∈Aπ(a∣s)(R(s,a)+γ∑s′∈SPss′aVπ(s′))(1)V_{\pi}(s) = \sum_{a \in \mathcal{A}} \pi(a \mid s) \Big( R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P_{ss'}^a V_{\pi}(s') \Big) \tag{1} Vπ​(s)=a∈A∑​π(a∣s)(R(s,a)+γs′∈S∑​Pss′a​Vπ​(s′))(1)

  1. 策略改进：计算 QQQ 值函数，π\piπ 会按照一定的方法根据 QQQ 值输出动作（如贪婪法，直接输出 QQQ 值最大的动作）：

Qπ(s,a)=R(s,a)+γ∑s′∈SPss′aVπ(s′)(2)Q_{\pi}(s, a) = R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P_{ss'}^a V_{\pi}(s') \tag{2} Qπ​(s,a)=R(s,a)+γs′∈S∑​Pss′a​Vπ​(s′)(2)

然后不断迭代上述两个步骤，这叫策略迭代（policy iteration）：

π0⇀评估Vπ0⇀改进π1⇀评估Vπ1⇀改进π2⇀评估⋯⇀改进π∗⇀评估V∗\pi_0 \xrightharpoonup{\text{评估}} V_{\pi_0} \xrightharpoonup{\text{改进}} \pi_1 \xrightharpoonup{\text{评估}} V_{\pi_1} \xrightharpoonup{\text{改进}} \pi_2 \xrightharpoonup{\text{评估}} \cdots \xrightharpoonup{\text{改进}} \pi_* \xrightharpoonup{\text{评估}} V_* π0​评估​Vπ0​​改进​π1​评估​Vπ1​​改进​π2​评估​⋯改进​π∗​评估​V∗​

# 自举

可以看到，需要用下一个状态 s′s's′ 的值函数来更新当前状态 sss 的值函数。但 s′s's′ 的值函数也是我们估算出来的，相当于要用一个估算去更新同类的估算，这种方法叫自举（bootstrapping）。

# 线性方程组的迭代解法

对于式 (1)(1)(1)，R(s,a)R(s, a)R(s,a)、Pss′aP_{ss'}^aPss′a​ 和 γ\gammaγ 已知，π\piπ 是给定的当前要评估的策略，也已知，唯一的未知数是状态值函数。那么式 (1)(1)(1) 可以看关于状态值函数的线性方程组。

线性方程组的数值求解包括直接法（如高斯消元) 和迭代解法，策略评估中采用了迭代解法中的高斯-赛德尔迭代法：

Vk+1(s)=∑a∈Aπ(a∣s)(R(s,a)+γ∑s′∈SPss′aVk(s′))V_{k+1}(s) = \sum_{a \in \mathcal{A}} \pi(a \mid s) \Big( R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P_{ss'}^a V_k(s') \Big) Vk+1​(s)=a∈A∑​π(a∣s)(R(s,a)+γs′∈S∑​Pss′a​Vk​(s′))

其中 kkk 表示第 kkk 次迭代。

# 策略评估

因此用迭代法进行策略评估的流程为：

  • Init V0(s)=0V_0(s) = 0V0​(s)=0
  • Repeat k=0,1,…k=0, 1, \dotsk=0,1,…:（kkk 是第 kkk 个高斯-赛德尔迭代）
    • for every s∈Ss \in \mathcal{S}s∈S do:（一次状态扫描）
      • Vk+1(s)=∑a∈Aπ(a∣s)(R(s,a)+γ∑s′∈SPss′aVk(s′))V_{k+1}(s) = \sum_{a \in \mathcal{A}} \pi(a \mid s) \Big( R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P_{ss'}^a V_k(s') \Big)Vk+1​(s)=∑a∈A​π(a∣s)(R(s,a)+γ∑s′∈S​Pss′a​Vk​(s′))
  • Until Vk+1(s)=Vk(s)V_{k+1}(s) = V_k(s)Vk+1​(s)=Vk​(s)

# 策略改进

这里把策略考虑为确定策略，即 π(s)=a\pi(s) = aπ(s)=a。那么一个很自然的方法就是用贪婪法（greedy）来更新策略，即直接输出 Q 值最大的动作：

π′(s)=arg⁡max⁡a∈AQπ(s,a)\pi'(s) = \arg \max_{a \in \mathcal{A}} Q_{\pi}(s, a) π′(s)=arga∈Amax​Qπ​(s,a)

那么把策略评估和策略改进合起来：

  • Init π0\pi_0π0​
  • Repeat t=0,1,…t = 0, 1, \dotst=0,1,…（ttt 是第 ttt 个时间步）
    • 进行策略评估，得到 VπtV_{\pi_t}Vπt​​
    • for every s∈Ss \in \mathcal{S}s∈S do:
      • πt+1(s)=arg⁡max⁡a∈AQπ(s,a)\pi_{t+1}(s) = \arg \max_{a \in \mathcal{A}} Q_{\pi}(s, a)πt+1​(s)=argmaxa∈A​Qπ​(s,a)
  • Until πk+1(s)=πk(s)\pi_{k+1}(s) = \pi_k(s)πk+1​(s)=πk​(s)

也就是有两个循环，内循环在用高斯-赛德尔迭代法进行策略评估，循环到值函数收敛为止；外循环在更新策略，循环到策略收敛为止。

# 最优性证明

证明更新后的 π′(s)\pi'(s)π′(s) 一定是更好的策略：

Qπ(s,π′(s))=max⁡a∈AQπ(s,a)≥Qπ(s,π(s))=Vπ(s)(3)Q_{\pi}(s, \pi'(s)) = \max_{a \in \mathcal{A}} Q_{\pi}(s, a) \geq Q_{\pi}(s, \pi(s)) = V_{\pi}(s) \tag{3} Qπ​(s,π′(s))=a∈Amax​Qπ​(s,a)≥Qπ​(s,π(s))=Vπ​(s)(3)

Vπ(s)≤Qπ(s,π′(s))=Eπ′[Rt+1+γVπ(st+1)∣St=s]≤Eπ′[Rt+1+γQπ(st+1,π′(st+1))∣St=s](代入式 3)≤Eπ′[Rt+1+γRt+2+γ2Qπ(st+2,π′(st+2))∣St=s](代入式 3)≤Eπ′[Rt+1+γRt+2+⋯∣St=s]=Vπ′(s)\begin{aligned} V_{\pi}(s) & \leq Q_{\pi}(s, \pi'(s)) = \mathbb{E}_{\pi'} \Big [ R_{t+1} + \gamma V_{\pi}(s_{t+1}) \mid S_t = s \Big ] \\ & \leq \mathbb{E}_{\pi'} \Big [ R_{t+1} + \gamma Q_{\pi} \big (s_{t+1}, \pi'(s_{t+1}) \big ) \mid S_t = s \Big ] \quad \text{(代入式 3)}\\ & \leq \mathbb{E}_{\pi'} \Big [ R_{t+1} + \gamma R_{t+2} + \gamma^2 Q_{\pi} \big (s_{t+2}, \pi'(s_{t+2}) \big ) \mid S_t = s \Big ] \quad \text{(代入式 3)}\\ & \leq \mathbb{E}_{\pi'} \Big [ R_{t+1} + \gamma R_{t+2} + \cdots \mid S_t = s \Big ] \\ & = V_{\pi'}(s) \end{aligned} Vπ​(s)​≤Qπ​(s,π′(s))=Eπ′​[Rt+1​+γVπ​(st+1​)∣St​=s]≤Eπ′​[Rt+1​+γQπ​(st+1​,π′(st+1​))∣St​=s](代入式 3)≤Eπ′​[Rt+1​+γRt+2​+γ2Qπ​(st+2​,π′(st+2​))∣St​=s](代入式 3)≤Eπ′​[Rt+1​+γRt+2​+⋯∣St​=s]=Vπ′​(s)​

因此策略更新到 π′\pi'π′ 后， VVV 和 QQQ 在状态 sss 的值都比更新前更高。

当更新停止后，有：

Qπ(s,π′(s))=max⁡a∈AQπ(s,a)=Qπ(s,π(s))=Vπ(s)Q_{\pi}(s, \pi'(s)) = \max_{a \in \mathcal{A}} Q_{\pi}(s, a) = Q_{\pi}(s, \pi(s)) = V_{\pi}(s) Qπ​(s,π′(s))=a∈Amax​Qπ​(s,a)=Qπ​(s,π(s))=Vπ​(s)

⇒Vπ(s)=max⁡a∈AQπ(s,a)\rArr V_{\pi}(s) = \max_{a \in \mathcal{A}} Q_{\pi}(s, a) ⇒Vπ​(s)=a∈Amax​Qπ​(s,a)

也就是对每个状态 sss，采取的都是最优动作，即这时 π\piπ 是最优策略。

# 值函数迭代

策略迭代会等到 VVV 收敛后再进行策略改进，而值函数迭代（value iteration）不等 VVV 收敛，而是评估一次后就进行策略改进：

  • Init π0,V0(s)=0\pi_0, V_0(s) = 0π0​,V0​(s)=0
  • Repeat t=0,1,…t = 0, 1, \dotst=0,1,…（ttt 是第 ttt 个时间步）
    • for every s∈Ss \in \mathcal{S}s∈S do
      • Vt+1(s)=max⁡a∈A(R(s,a)+γ∑s′∈SPss′aVt(s′))V_{t+1}(s) = \max_{a \in \mathcal{A}} \Big( R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P_{ss'}^a V_t(s') \Big)Vt+1​(s)=maxa∈A​(R(s,a)+γ∑s′∈S​Pss′a​Vt​(s′))
  • Until Vt+1(s)=Vt(s)V_{t+1}(s) = V_t(s)Vt+1​(s)=Vt​(s)
  • Output π(s)=arg⁡max⁡a∈A(R(s,a)+γ∑s′∈SPss′aVt(s′))\pi(s) = \arg \max_{a \in \mathcal{A}} \Big( R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P_{ss'}^a V_t(s') \Big)π(s)=argmaxa∈A​(R(s,a)+γ∑s′∈S​Pss′a​Vt​(s′))

可以看到值函数迭代过程中并没有一个显示的策略。
