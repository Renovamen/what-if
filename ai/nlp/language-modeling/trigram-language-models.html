# 三元语言模型

三元（Trigram）语言模型是马尔科夫模型在语言建模问题上的直接应用。本节讨论三元模型的基本定义、极大似然估计和优缺点。

# 基本定义

按照二阶马尔科夫模型：

P(X1=x1,X2=x2,...,Xn=xn)P(X_1 = x_1, X_2 = x_2, ... , X_n = x_n) P(X1​=x1​,X2​=x2​,...,Xn​=xn​)

=∏i=1nP(Xi=xi∣Xi−2=xi−2,Xi−1=xi−1)= \prod_{i=1}^n P(X_i=x_i \mid X_{i-2}=x_{i-2}, X_{i-1}=x_{i-1}) =i=1∏n​P(Xi​=xi​∣Xi−2​=xi−2​,Xi−1​=xi−1​)

其中，我们假设：

P(Xi=xi∣Xi−2=xi−2,Xi−1=xi−1)=q(xi∣xi−2,xi−1)P(X_i=x_i \mid X_{i-2}=x_{i-2}, X_{i-1}=x_{i-1}) = q(x_i \mid x_{i-2}, x_{i-1}) P(Xi​=xi​∣Xi−2​=xi−2​,Xi−1​=xi−1​)=q(xi​∣xi−2​,xi−1​)

对任意 (u,v,w)(u,v,w)(u,v,w)，q(w∣u,v)q(w \mid u,v)q(w∣u,v) 是模型的参数，之后我们将会讨论如何从训练集中估计出参数。则模型现在可以写为：

p(x1...xn)=∏i=1nq(xi∣xi−2,xi−1)p(x_1 ... x_n)=\prod_{i=1}^n q(x_i \mid x_{i-2}, x_{i-1}) p(x1​...xn​)=i=1∏n​q(xi​∣xi−2​,xi−1​)

则定义 1.2（三元语言模型）：

三元语言模型由一个有限集 V\mathcal{V}V 和一个参数 q(w∣u,v)q(w \mid u,v)q(w∣u,v) 组成，其中 w∈V∪{STOP}w \in \mathcal{V} \cup \{\text{STOP}\}w∈V∪{STOP} ，u,v∈V∪{∗}u,v \in \mathcal{V} \cup \{*\}u,v∈V∪{∗} 。q(w∣u,v)q(w \mid u,v)q(w∣u,v) 可以理解为单词 www 正好出现在二元组（Bigram） (u,v)(u,v)(u,v) 之后的概率。三元语言模型中任意句子 x1...xnx_1 ... x_nx1​...xn​ （x0=x−1=∗x_0 = x_{-1} = *x0​=x−1​=∗）出现的概率为：

p(x1...xn)=∏i=1nq(xi∣xi−2,xi−1)p(x_1 ... x_n)=\prod_{i=1}^n q(x_i \mid x_{i-2}, x_{i-1}) p(x1​...xn​)=i=1∏n​q(xi​∣xi−2​,xi−1​)

例如，对于句子：the dog barks STOP，有：

p(the dog barks STOP)p(\text{the dog barks STOP}) p(the dog barks STOP)

=q(the∣∗,∗)×q(dog∣∗,the)×q(barks∣the, dog)×q(STOP∣dog, barks)=q(\text{the} \mid *,*) \times q(\text{dog} \mid *,\text{the}) \times q(\text{barks} \mid \text{the, dog}) \times q(\text{STOP} \mid \text{dog, barks}) =q(the∣∗,∗)×q(dog∣∗,the)×q(barks∣the, dog)×q(STOP∣dog, barks)

可以看到每个单词只依赖于它的前两个单词（三元假设（Trigram Assumption））。

参数 q(w∣u,v)q(w \mid u,v)q(w∣u,v) 也可以被理解为在给定上文 u,vu,vu,v 的条件下，单词 www 的概率分布。它需要满足以下条件：

  • 对任意三元组 u,v,wu,v,wu,v,w，q(w∣u,v)≥0q(w \mid u,v) \geq 0q(w∣u,v)≥0；

  • 对任意二元组 u,vu,vu,v，∑w∈V∪{STOP}q(w∣u,v)=1\sum_{w \in \mathcal{V} \cup \{\text{STOP}\}} q(w \mid u,v) = 1∑w∈V∪{STOP}​q(w∣u,v)=1；


那么现在的关键问题就是该如何通过训练集估计出模型的参数 q(w∣u,v)q(w \mid u,v)q(w∣u,v)。其中 www 可能是 V∪{STOP}\mathcal{V} \cup \{\text{STOP}\}V∪{STOP} 中的任何元素，u,vu,vu,v 可能是 V∪{∗}\mathcal{V} \cup \{*\}V∪{∗} 中的任何元素，所以模型的参数会有 ∣V∣3|\mathcal{V}|^3∣V∣3 个，这个数字很可能非常庞大。

# 极大似然估计

定义 c(u,v,w)c(u,v,w)c(u,v,w) 为三元组 (u,v,w)(u, v, w)(u,v,w) 在训练集中出现的次数，如 c(the, dog, barks)c(\text{the, dog, barks})c(the, dog, barks) 为单词序列 the dog barks 在训练集中出现的次数。定义 c(u,v)c(u,v)c(u,v) 为二元组 (u,v)(u,v)(u,v) 在训练集中出现的次数。对任意 w,u,vw,u,vw,u,v，它的极大似然估计（Maximum-likelihood Parameter Estimates）为：

q(w∣u,v)=c(u,v,w)c(u,v)q(w \mid u,v)=\frac{c(u,v,w)}{c(u,v)} q(w∣u,v)=c(u,v)c(u,v,w)​

比如我们要估计 q(barks∣the, dog)q(\text{barks} \mid \text{the, dog})q(barks∣the, dog)：

q(barks∣the, dog)=c(the, dog, barks)c(the, dog)q(\text{barks} \mid \text{the, dog})=\frac{c(\text{the, dog, barks})}{c(\text{the, dog})} q(barks∣the, dog)=c(the, dog)c(the, dog, barks)​

这是一种很自然的估计方式：如果要估计 barksbarksbarks 出现在 (the, dog)(\text{the, dog})(the, dog) 后的概率，那么计算一下 (the, dog)(\text{the, dog})(the, dog) 出现了多少次，再计算一下 (the, dog, barks)(\text{the, dog, barks})(the, dog, barks) 出现了多少次，然后算这两个数的比例就行。但极大似然估计有两个很严重的问题：

  1. 如果一个三元组在训练集中没有出现，那么 q(w∣u,v)=0q(w \mid u,v)=0q(w∣u,v)=0（因为分子为 0）。由于参数规模一般会很大，这种情况会经常出现，导致数据很稀疏。而且这是不合理的，一个三元组在训练集中没有不出现不等于它出现的概率为 0；
  2. 分母 c(u,v)c(u,v)c(u,v) 也有可能为 0，这时这个估计的定义就不合法了。

后面我们会讨论该如何改进参数估计方法来解决这些问题，但现在我们先讨论该如何评估一个语言模型的好坏。

# 困惑度

语言模型评估指标：困惑度（Perplexity）

# 定义

假设测试集中有 mmm 个句子 x(1),x(2),...,x(m)x^{(1)},x^{(2)}, ... ,x^{(m)}x(1),x(2),...,x(m)，每个测试句子 x(i)(i∈{1...m})x^{(i)} \ (i \in \{1 ... m\})x(i)(i∈{1...m}) 是一个单词序列 x1(i),...,xni(i)x_1^{(i)},...,x_{n_i}^{(i)}x1(i)​,...,xni​(i)​，nin_ini​ 是第 iii 个句子的长度，每个句子都以 STOP 符号结尾。

我们可以用语言模型测出每个测试句子 x(i)x^{(i)}x(i) 出现的概率 p(x(i))p(x^{(i)})p(x(i))。一个容易想到的评估指标是该模型测出的所有测试句子出现的概率 ∏i=1mp(x(i))\prod_{i=1}^m p(x^{(i)})∏i=1m​p(x(i))，毕竟质量越高的模型处理测试句子的能力越强。


模型在测试集上的 perplexity（困惑度）是这个指标的变形。

令 M=∑i=1mniM=\sum_{i=1}^m n_iM=∑i=1m​ni​ 为测试集中的单词总数。 对 ∏i=1mp(x(i))\prod_{i=1}^m p(x^{(i)})∏i=1m​p(x(i)) 取对数再除以 MMM，有：

1Mlog⁡2∏i=1mp(x(i))=1M∑i=1mlog⁡2p(x(i))\frac{1}{M}\log_2 \prod_{i=1}^m p(x^{(i)}) = \frac{1}{M} \sum_{i=1}^m \log_2 p(x^{(i)}) M1​log2​i=1∏m​p(x(i))=M1​i=1∑m​log2​p(x(i))

定义 1.3（perplexity）：

perplexity=2−l\text{perplexity} = 2^{-l} perplexity=2−l

其中：

l=1M∑i=1mlog⁡2p(x(i))l=\frac{1}{M} \sum_{i=1}^m \log_2 p(x^{(i)}) l=M1​i=1∑m​log2​p(x(i))

perplexity 是一个正数，perplexity 越小，模型处理测试句子的能力就越强。

    其中二元模型中每个单词只依赖于它的前一个单词：

# 优缺点

三元假设太苛刻了，且在语义上过于简单，但它有较好实际应用效果。
