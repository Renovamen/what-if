# 对数线性模型

对数线性模型（Log-Linear Models）在 NLP 中的应用非常广泛，它的主要优点是能够利用丰富的特征集，从而相比前面提到的简单的评估方法能更好的表示数据。本章将讨论其定义和参数估计的方法。这里有两个引出对数线性模型的例子：

# Language Modeling

考虑一个语言模型，任务是对以下条件概率进行估计：

P(Wi=wi∣W1=w1…Wi−1=wi−1)=p(wi∣w1…wi−1)P(W_i=w_i \mid W_1 = w_1 \dots W_{i-1} = w_{i-1}) = p(w_i \mid w_1 \dots w_{i-1}) P(Wi​=wi​∣W1​=w1​…Wi−1​=wi−1​)=p(wi​∣w1​…wi−1​)

也就是在给定前面的单词序列 w1…wi−1w_1 \dots w_{i-1}w1​…wi−1​ 的条件下，对 wiw_iwi​ 的分布进行建模。

在三元模型中，我们假设了：

p(wi∣w1…wi−1)=p(wi∣wi−2,wi−1)=q(wi∣wi−2,wi−1)p(w_i \mid w_1 \dots w_{i-1})=p(w_i \mid w_{i-2},w_{i-1})=q(w_i \mid w_{i-2},w_{i-1}) p(wi​∣w1​…wi−1​)=p(wi​∣wi−2​,wi−1​)=q(wi​∣wi−2​,wi−1​)

q(w∣u,v)q(w \mid u,v)q(w∣u,v) 是模型的参数，可以通过一些方法估计出来（如极大似然估计）。

现在我们想估计下一个单词 wiw_iwi​ 是 model 的概率，即：

P(Wi=model∣W1=w1…Wi−1=wi−1)P(W_i=\text{model} \mid W_1=w_1 \dots W_{i-1}=w_{i-1}) P(Wi​=model∣W1​=w1​…Wi−1​=wi−1​)

我们希望利用上下文中的所有特征来估计 model 出现在下一个单词的概率，而不是像三元模型那样仅仅利用前两个单词。如：只考虑单词 wi−2w_{i-2}wi−2​ 的影响：

P(Wi=model∣Wi−2=any)P(W_i = \text{model} \mid W_{i-2} = \text{any}) P(Wi​=model∣Wi−2​=any)

考虑前一个词是形容词（adjective）的影响（pos\text{pos}pos 是把单词映射到其词性的函数）：

P(Wi=model∣pos(Wi−1)=adjective)P(W_i = \text{model} \mid \text{pos}(W_{i-1}) = \text{adjective}) P(Wi​=model∣pos(Wi−1​)=adjective)

考虑前一个词的后缀是 ical 的影响（suff4\text{suff4}suff4 是把单词映射到其最后四个字母的函数）：

P(Wi=model∣suff4(Wi−1)=ical)P(W_i = \text{model} \mid \text{suff4}(W_{i-1}) = \text{ical}) P(Wi​=model∣suff4(Wi−1​)=ical)

考虑单词 word 从未出现在上下文中过的影响：

P(Wi=model∣Wj≠model for j∈1…(i−1))P(W_i = \text{model} \mid W_j \not= \text{model for } j \in {1 \dots (i-1)}) P(Wi​=model∣Wj​=model for j∈1…(i−1))

考虑单词 grammatical 在上下文中出现过的影响：

P(Wi=model∣Wj=grammatical for some j∈1…(i−1))P(W_i = \text{model} \mid W_j = \text{grammatical for some } j \in {1 \dots (i-1)}) P(Wi​=model∣Wj​=grammatical for some j∈1…(i−1))

一个简单粗暴的方法是魔改一下线性插值法，把上述估计直接按不同的权重合在一起：

p(model∣w1…wi−1)=λ1×qML(model∣wi−2=any,wi−1=statistical)+λ2×qML(model∣wi−1=statistical)+λ3×qML(model)+λ4×qML(model∣wi−2=any)+λ5×qML(model∣wi−1 is an adjective)+λ6×qML(model∣wi−1 ends in “ical”)+λ7×qML(model∣“model” does not occur somewhere in w1…wi−1)+λ8×qML(model∣“grammatical” occurs somewhere in w1…wi−1)+…\begin{aligned} p(\text{model} \mid w_1 \dots w_{i-1}) &= \lambda_1 × q_{ML}(\text{model} \mid w_{i-2} = \text{any}, w_{i-1} = \text{statistical}) \\ &+ \lambda_2 × q_{ML}(\text{model} \mid w_{i-1} = \text{statistical}) \\ &+ \lambda_3 × q_{ML}(\text{model}) \\ &+ \lambda_4 × q_{ML}(\text{model} \mid w_{i-2} = \text{any}) \\ &+ \lambda_5 × q_{ML}(\text{model} \mid w_{i-1} \text{ is an adjective}) \\ &+ \lambda_6 × q_{ML}(\text{model} \mid w_{i-1} \text{ ends in “ical”}) \\ &+ \lambda_7 × q_{ML}(\text{model} \mid \text{“model” does not occur somewhere in } w_1 \dots w_{i-1}) \\ &+ \lambda_8 × q_{ML}(\text{model} \mid \text{“grammatical” occurs somewhere in } w_1 \dots w_{i-1}) \\ &+ \text{ } \dots \end{aligned} p(model∣w1​…wi−1​)​=λ1​×qML​(model∣wi−2​=any,wi−1​=statistical)+λ2​×qML​(model∣wi−1​=statistical)+λ3​×qML​(model)+λ4​×qML​(model∣wi−2​=any)+λ5​×qML​(model∣wi−1​ is an adjective)+λ6​×qML​(model∣wi−1​ ends in “ical”)+λ7​×qML​(model∣“model” does not occur somewhere in w1​…wi−1​)+λ8​×qML​(model∣“grammatical” occurs somewhere in w1​…wi−1​)+…​

但当要考虑的条件特别多时，实现这种方法会非常困难。而对数线性模型是一种更合适的能利用更多上下文信息的方法。

# POS Tagging

考虑单词序列 wi…wnw_i \dots w_nwi​…wn​ 和其对应的标签序列 t1…ti−1t_1 \dots t_{i-1}t1​…ti−1​（i<ni<ni<n），我们的任务是对第 iii 个标签的条件分布进行建模，即：

P(Ti=ti∣T1=t1…Ti−1=ti−1,W1=w1…Wn=wn)P(T_i = t_i \mid T_1 = t_1 \dots T_{i-1} = t_{i-1}, W_1 = w_1 \dots W_n = w_n) P(Ti​=ti​∣T1​=t1​…Ti−1​=ti−1​,W1​=w1​…Wn​=wn​)

词性标注也有很多对估计 tit_iti​ 的分布有用的上下文信息，简单粗暴的魔改线性插值法依然会在要考虑的条件很多时面临困难，所以这时依然需要对数线性模型。
