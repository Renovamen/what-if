# 参数估计

# Log-Likelihood Function

假设有一个有样本 (x(i),y(i)) 组成的训练集（i∈{1...n},x(i)∈X,y(i)∈Y）。直观上来说，对任意样本 i，logp(y(i)∣x(i);v) 越大说明模型越能拟合这个样本。所以 log-likelihood 就是每个样本的条件概率取对数的和：

L(v)=i=1∑n​logp(y(i)∣x(i);v)(3.3)

这是一个关于参数向量 v 的函数，可以理解为是评估 v 对训练数据的拟合程度的函数。

 

现在考虑用极大似然估计来进行参数估计，即：

vML​=argv∈Rdmax​L(v)

但极大似然估计会存在一些问题，尤其是在特征数量很大的时候。比如考虑一个只有一元、二元、三元特征的语言模型。三元组 any statistical model 只在训练集的第 100 个样本中出现过，即：

fN(any, statistical, model)​(x(100),y(100))=1

同时这是训练集中唯一满足 u=any,v=statistical 的三元组 (u,v,w)。

在这种情况下，估计出来的参数会为 +∞，因为对第 100 个样本，有：

v⋅f(x(100),y(100))=k=1∑d​vk​fk​(x(100),y(100))

=v1​fN(any, statistical, model)​(x(100),y(100))+v2​fN(statistical, model)​(x(100),y(100))+v3​fN(model)​(x(100),y(100))

=v1​

 

p(y∣x;v)=∑y′∈Y​exp(v⋅f(x,y′))exp(v⋅f(x,y))​(3.1)

p(y(100)∣x(100);v)=∑y′∈Y​exp(v⋅f(x(100),y′))exp(v⋅f(x(100),y(100)))​=1

⇒∑y′∈Y​exp(v⋅f(x(100),y′))exp(v1​)​=1

⇒∑y′∈Y​exp(v⋅f(x,y′))exp(v⋅f(x,y))​

⇒v→+∞

这时不管其他样本的条件概率怎么样，L(v) 都会最大。这样估计出来的参数在测试集上的表现会很差（过拟合），它会由“训练集 any statistical 的后面一定会跟着 model”的现象而直接得出 P(Wi​=model∣Wi−1​,Wi−2​=any, statistical))=1 的结论。

# 正则化（Regularization）

一个常用的解决方案是在公式 3.3 中加入一个正则化项（regularization term）来避免参数的估计过大（趋近于无穷）。一个常用的正则化项是参数的 2 范数（2-norm），即：

∥v∥2=k∑​vk2​

其中 ∥v∥ 是向量 v 的长度（欧几里得范数（Euclidean norm），因为 ∥v∥=∑k​vk2​​，也即 L2 范数）。
