# 定义

# 基本定义

对数线性模型要解决的问题是：有输入集合 X 和标签集合 Y，对条件概率 p(y∣x)（x∈X,y∈Y）进行建模。

如在语言模型中，Y 是词典 V，X 是单词序列 w1​…wi−1​（i≥1,wj​∈V,j∈{1...(i−1)}）的集合。输入 x 为 w1​…wi−1​，标签 y 为 wi​。

在词性标注中， Y 是标签集合 T，X 是上下文 ⟨w1​w2​…wn​,t1​t2​…ti−1​⟩ 的集合（n≥1,wi​∈V,i∈{1…n},tj​∈T,j∈{1…(i−1)}）。

 

定义 3.1（对数线性模型）：

对数线性模型由以下部分组成：

  • 可能的输入集合 X；
  • 可能的标签集合 Y，假设它为有限集；
  • 一个正整数 d，它确定了模型中的特征和参数数量；
  • 一个函数 f:X×Y→Rd，它把 (x,y) 映射到了其特征向量 f(x,y)；
  • 一个参数向量 v∈Rd

对任意 x∈X,y∈Y，该模型定义了以下条件概率：

p(y∣x;v)=∑y′∈Y​exp(v⋅f(x,y′))exp(v⋅f(x,y))​(3.1)

其中，exp(x)=ex，v⋅f(x,y)=∑k=1d​vk​fk​(x,y) 是 v 和 f(x,y) 的内积（inner product）。p(y∣x;v) 可以被理解为：当参数为 v 时，在给定 x 的条件下 y 的概率。

# 模型形式

公式 3.1 中，最主要的是求内积 v⋅f(x,y) 。 v⋅f(x,y) 越大说明在给定上下文 x 的情况下，y 出现的概率越大。v⋅f(x,y) 可以是任意实数，但 exp(v⋅f(x,y)) 只可能是正数。

 

公式 3.1 的分母是一个归一化项（normalization term），用于保证分布的和为 1，即：

y∈Y∑​p(y∣x;v)=1

 

将公式 3.1 拆成以下形式：

g(x)=logy′∈Y∑​exp(v⋅f(x,y′))

logp(y∣x;v)​=v⋅f(x,y)−logy′∈Y∑​exp(v⋅f(x,y′))=v⋅f(x,y)−g(x)​(3.2)

公式 3.2 中的第一项 v⋅f(x,y) 对 f(x,y) 是线性的，而第二项 g(x) 只与 x 有关，与 y 无关。所以当 x 一定时，取对数后的概率 logp(y∣x;v) 对 f(x,y) 是线性的，所以这个模型叫“对数线性模型”。
